Wordle Clone

A clean and responsive web-based recreation of the popular word-guessing game Wordle. It challenges players to identify a hidden 5-letter word within six attempts using logic and vocabulary. This project also features seasonal "Easter eggs," such as a special Valentine's Mode that asks users a very important question if they win!

Key Features

  • Classic Grid System: A standard 5x6 grid with fluid feedback animations.
  • Seasonal Iterations: Custom versions for holidays (Valentines, Christmas, etc.) to keep the experience fresh.
  • Unlimited Play: Unlike the original, you can play as many rounds as you want.
  • Speed Build: Developed entirely from scratch in under 5 hours as a personal coding challenge.

Use Cases

Brain training and casual gaming. It stands as a testament to rapid prototype development and creative UI/UX experimentation.
